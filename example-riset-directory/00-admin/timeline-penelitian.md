## Contingency Plan

### Jika Responden <25
- Plan A: Extend recruitment period 1 week
- Plan B: Lower n to 20 (still adequate untuk paired t-test dengan power 0.8)

### Jika Data Tidak Signifikan
- Conduct failure analysis (boundary conditions)
- Report negative results + deep insight
- Shift contribution ke methodological framework

### Jika Timeline Delay >2 weeks
- Skip submission ke konferensi, fokus ke jurnal (lebih fleksibel deadline)
- Request extension dari pembimbing
