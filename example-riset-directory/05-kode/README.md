# 05-kode

Source code untuk analisis data dan visualisasi — skrip Python untuk statistical analysis.

## Struktur yang direncanakan

```
05-kode/
├── analysis/              # Skrip analisis statistik
│   ├── descriptive_stats.py
│   ├── normality_test.py
│   ├── paired_ttest.py
│   ├── sus_calculation.py
│   └── visualization.py
├── utils/                 # Helper functions
│   ├── data_loader.py
│   └── report_generator.py
└── requirements.txt       # Python dependencies
```

## Dependencies

- pandas
- numpy
- scipy
- matplotlib
- seaborn
- statsmodels
