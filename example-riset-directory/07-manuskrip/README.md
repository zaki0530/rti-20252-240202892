# 07-manuskrip

Draf naskah ilmiah untuk publikasi — **target jurnal HCI atau konferensi UI/UX**.

## Naskah konsolidasi

- [naskah-jurnal.md](naskah-jurnal.md) — naskah lengkap dalam format IMRAD (Judul/Penulis, Abstrak ID+EN, Introduction – Conclusion, References)
- [naskah-jurnal.pdf](naskah-jurnal.pdf) — versi PDF untuk submission

## Struktur naskah per bagian (sumber/draf kerja)

- [00-outline.md](00-outline.md) — outline paper dan consistency matrix
- [01-abstract.md](01-abstract.md) — Abstrak (ID & EN)
- [02-introduction.md](02-introduction.md) — Introduction (context, gap, research questions, contributions)
- [03-related-work.md](03-related-work.md) — Related Work (UI/UX mobile apps, usability metrics, F&B app evaluation)
- [04-method.md](04-method.md) — Method (within-subject design, participants, tasks, metrics, procedures, analysis)
- [05-results.md](05-results.md) — Results (descriptive stats, normality test, paired t-test results)
- [06-discussion.md](06-discussion.md) — Discussion (interpretation, comparison with benchmarks, limitations)
- [07-conclusion.md](07-conclusion.md) — Conclusion & Future Work
- [08-references.md](08-references.md) — References (format sesuai target publikasi)

## Target Publikasi

- **Jurnal Nasional Terakreditasi**: Jurnal Ilmu Komputer, JNTETI, JUTI
- **Konferensi**: ICAICTA, ICITISEE, atau konferensi HCI regional
- **Jurnal Internasional** (jika hasil sangat baik): International Journal of Human-Computer Interaction, Behaviour & Information Technology
