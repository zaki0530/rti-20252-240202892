# 00-admin — Administrasi Penelitian

Folder ini berisi dokumen administratif dan kronologis pelaksanaan penelitian.

## Isi Folder

- **jadwal-dan-log-penelitian.md** — Log kronologis pelaksanaan Tahap 1 hingga Tahap 5, status progres, dan checklist item tindak lanjut
- **timeline-penelitian.md** — (Opsional) Diagram Gantt atau timeline visual pelaksanaan penelitian

## Catatan

Semua tanggal pelaksanaan mengacu pada progres aktual yang tercatat dalam riwayat commit Git dan dokumen tahapan di folder `09-docs/`.
